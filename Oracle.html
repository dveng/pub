SQL & DQL

结构化查询语言(Structured Query Language)简称SQL.
SQL语言共分为四大类：数据查询语言DQL，数据操纵语言DML，数据定义语言DDL，数据控制语言DCL。
DQL:Data QueryLanguage 数据查询语言,

  • 1.数据查询语言DQL基本结构是由SELECT子句，FROM子句，WHERE
    子句组成的查询块：
    SELECT <字段名表>
    FROM <表或视图名>
    WHERE <查询条件>

  • 2 .数据操纵语言DML
    数据操纵语言DML主要有三种形式：
    1) 插入：INSERT
    2) 更新：UPDATE
    3) 删除：DELETE

  • 3.数据定义语言DDL
    数据定义语言DDL用来创建数据库中的各种对象—–表、视图、
    索引、同义词、聚簇等如：
    CREATE TABLE/VIEW/INDEX/SYN/CLUSTER
    | | | | |
    表 视图 索引 同义词 簇
    DDL操作是隐性提交的！不能rollback
    4.更加详尽ALTER,CREATE,DROP,QUIT,REVOKE,GRANT等.

什么是数据库?

数据库是一种存储结构. 它允许使用各种格式输入\处理和检索数据,不必在每次需要数据时重新输入.
Database(数据库): 相互之间有关联关系的Table的集合
例如:图书管理数据库系统
表的集(Table1:出版社.T2:出版社图书目录.T3:采买记录.T4:图书.T5:借阅登记…)合集成为数据库.

create table user_account(

       user_id number(10) primary key,

       user_name varchar2(20) check(length(user_name)>3) not null unique,

       user_password number(13) check(length(user_password)>=8 and length(user_password)<=13),

       user_balance number(20,6) default 0.0 check((user_balance)>=0.0 and (user_balance) is not null)

);
